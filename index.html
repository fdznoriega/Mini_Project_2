Hosting Experience

In creating this visualization, we created a calculated field called “Host Experience (Month)” which counts the number of months by subtracting today’s date from “Host Since”. From the trend line added to the scatterplot, it’s obvious that there is a generally positive relationship between the host experience and number of visits measured by the attribute “number of reviews.” However, we can tell from the R-squared value that the predictability of this model here is extremely weak.

Reflection

Overview

We analyzed the dataset of Airbnb Listings (30,478 entries) in New York City on Tableau Public which was compiled from Inside Airbnb. We chose this dataset because it's relatively clean and purposeful enough for us to come up with meaningful questions. It’s important to note that this dataset was sourced from only publicly available information on the Airbnb website and had already been cleansed and aggregated before being adopted by Tableau Public.

Data Selection

While looking for an interesting yet meaningful dataset to analyze, we found a couple candidates including this Airbnb dataset, polls on the 2020 election, and the 2016 presidential candidate expenditures. The Airbnb dataset was the cleanest, most complete, and easiest to analyze one since the attributes all very closely related to the topic: Airbnb listings. Airbnb is an easy topic to understand by the audience and since we all have experience with living at an Airbnb, it’s relatively easy for us to come up with not only more questions, but also meaningful ones that we can try to answer with analyzing the dataset.

For example, the entire “party” attribute of the presidential candidate expenditure dataset is empty and therefore the whole dataset was unable to be analyzed.

Processing

We noticed that there are many null values for the “Review Scores Rating (bin)” and “Review Scores Rating” attributes. Instead of preprocessing them, we chose to wait until after we came up with the visualizations and used the filter and exclude functions in Tableau to get rid of those values. Additionally, we realized that the values for “Name” are descriptions of the listings and are therefore unable to be used in the exploration stage that followed.
